叶片如何接入主版本

叶片不会改写花朵计划、花朵随机数或主要构图。它从同一根实际花茎读取成员身份，再以独立随机数生成已确认的叶型和叶序。

打开 4 组 3D 搭配 LAB打开 25 种花型总览
01 Flower plan主版本决定今天出现哪些花，不因叶片系统改变。
02 PlantStemInstance每朵花、花茎与成员 ID 绑定到同一实例。
03 Foliage profile只查已确认映射；未研究成员返回 unresolved + none。
04 Independent RNG叶片使用独立 seed，不扰动原有花朵位置与配色。
05 Geometry + order按 Strap / Palmate 与 basal / alternate 生成。
06 Ownership audit检查孤立叶、错茎、多 profile 和遮挡风险。
