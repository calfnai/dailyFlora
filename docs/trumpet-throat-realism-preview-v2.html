v2 错误实验记录

这个页面用于回答“v2 到底做了什么”。v2 不是有效结果：它为了增加拖拽旋转，重新用 Canvas 2.5D 投影画了一套花，导致花型偏离 v1。它应作为失败样本保留，不作为后续实现依据。

v2 做错的事

它不是“给 v1 加旋转”

  • 把 v1 的 SVG 正面画法重写成 Canvas 投影。
  • 为了做 yaw / pitch / zoom，重新定义了花瓣、cup、茎叶的空间坐标。
  • 因此花型不再等同于 v1，只是一个新的实验花。
  • Pages 里旧 v2 地址没有稳定发布，导致你打开 404。
正确下一步

v3 的目标

  • 左侧固定显示 v1 正面基准。
  • 右侧只在同一套花型上增加拖拽旋转。
  • 重置视角时，应尽量贴近 v1 正面。
  • 旋转只用于查看 cup / corona 的突出关系，不重新定义花。
v2 技术变化

可追溯变化

v2 对应源码路径：

docs/trumpet-throat-realism-preview.html

它引入了：

  • Canvas 绘制
  • pointer drag 旋转
  • wheel zoom
  • yaw / pitch / zoom 参数显示
验收结论

v2 不合格

用户反馈“比上一版更奇怪”，原因是我把交互需求误读为重新建模需求。v2 只保留为错误记录，避免以后重复这个方向。

后续不要继续修 v2 的花型。正确路线是以 v1 为视觉基准，做 v3 / v4：只增加 360 拖拽查看，不改花瓣、cup、rim、喉部和绿色连接点的正面基准。
