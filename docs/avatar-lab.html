DailyFlora Avatar Lab

用单朵花 primitive 做一套稳定、可识别、像 GitHub identicon 一样的默认头像体系。

3D 头像框

默认头像直接作为 WebGL 头像框使用：列表里可以低帧静止，资料页或 hover 时轻微旋转。

头像体系规则

seed 决定花型、调色盘、背景和旋转节奏。 flower 用 primitive lab 的单朵花，避免通用头像感。 ring 和背景色提供 GitHub 式的账号区分度。 motion 是轻微呼吸和旋转，静态导出保持同一身份。
