从照片提取的建模约束

观察项正确目标
开放区约占有效花序下部 40–50%；花朵可独立辨认，不封成表皮。
过渡区中部约 20–30%；花被半开、直径递减，形成明确渐变。
花蕾区顶部约 20–30%；花梗最短，轮廓逐渐收尖。
雄蕊六枚，略长于花被或与之接近；沿每朵花的局部法线伸出，不组成水平圆环。
空气感每圈保留轴向错位与空隙；密度来自连续层级，不来自同高度堆叠。

最多两个几何方向

方案 A：清晰器官优先建议
12–18 朵可读开放花 + 6–10 朵半开花 + 顶端花蕾；沿主轴错落旋生。每朵花独立包含六片花被、花心、六枚克制雄蕊、花药与短花梗。先确保侧视和顶视都能读出单花。
方案 B：分层密度优先
前景 / 外轮廓保留少量完整单花，内层使用更简化的半开花与花蕾补足连续感；仍维持下长上短花梗和三段开放梯度。更接近密穗，但较容易再次形成“橙色墙”。
比例为本项目的视觉目标区间，不是所有 Eremurus 物种的植物学常数。
